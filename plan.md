# Digital Marketing & Web Designer Portfolio Plan

Transform the existing profile into a vibrant, creative, and witty portfolio for a Digital Marketing & Web Designer.

## Scope Summary
- **Witty Tone**: Infuse humor and personality into all copy (Hero, About, Contact).
- **Data Update**: Revamp `src/data/profile.ts` to include:
  - Digital marketing & design services.
  - Project metrics (e.g., "Increased ROI by 300%").
  - Client testimonials with avatars.
  - Creative witty bio and project descriptions.
- **Component Enhancements**:
  - `Projects.tsx`: Add metrics badges and testimonial overlays or cards.
  - `Hero.tsx`: Dynamic, fun layout with creative taglines.
  - `About.tsx`: "Not-so-boring" bio section.
  - `Contact.tsx`: Witty form labels and witty success messages.
- **Visuals**: Use vibrant colors and creative image placeholders.

## Phase 1: Data Revamp
- Update `profileData` in `src/data/profile.ts` with:
  - New images from recent generation.
  - Metrics for each project.
  - A `testimonials` array.
  - Witty descriptions.

## Phase 2: Component Overhaul
- **Navbar**: Subtle creative touches.
- **Hero**: Update text to be witty and bold.
- **About**: Personality-first approach.
- **Projects**: Redesign project cards to highlight metrics and testimonials.
- **Experience**: Marketing-focused roles with creative descriptions.
- **Contact**: Fun form prompts.

## Phase 3: Polish & Theme
- Ensure the color palette feels "creative" (maybe some gradient accents).
- Verify mobile responsiveness.
- Check accessibility.
